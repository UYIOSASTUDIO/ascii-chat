This system provides an ephemeral, end-to-end encrypted communication channel. No logs. No trace.

1. CONNECTION PROTOCOL

  • Every session generates a unique, temporary Session Key.
  • Encryption: ECDH P-256 + AES-GCM (256-bit).
  • Keys are generated locally in your browser. The server never sees them.

2. COMMAND LIST

/connect [KEY]
Initiates a secure handshake with another user using their Session Key.

/ping [NAME]
Scans the network for active users with a specific name. Returns their Session Key.

/info [KEY]
Runs a deep trace on a specific Key. Reveals IP status, OS fingerprint, and login time.

/ghost
Toggles Stealth Mode. Makes you invisible to network scans (/ping).

/clear
Wipes the terminal screen output.

3. SECURITY WARNINGS

This system is protected against Man-in-the-Middle attacks via SAS Fingerprinting.
Always compare the "Digital Fingerprint" shown at chat start with your partner via a secondary channel.
